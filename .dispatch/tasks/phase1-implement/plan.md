- [x] Create `src/render/patterns.ts` that generates SVG `<defs>` for: dotgrid (8x8 grid with 1px green dots), crosshatch (8x8 diagonal lines), hero (6x6 green bg with subtle dots). Each pattern must be reusable via `fill="url(#patternName)"`.
